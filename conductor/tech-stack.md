# Tech Stack: Shared AI Storytelling RPG

## 1. Programming Languages
- **Python**: Primary language for backend logic and AI integration.
- **TypeScript**: Used for building the React frontend for strong typing and better tooling.

## 2. Frameworks
- **FastAPI (Python)**: High-performance, modern web framework for building APIs with Python.
- **React (TypeScript)**: Popular frontend library for building dynamic user interfaces and story sessions.

## 3. Storage & Database
- **Firebase Firestore**: Serverless, NoSQL document database for managing world state, player profiles, and session data.
- **Structured JSON**: Used for world-state deltas and narrative state snapshots.

## 4. AI & External Integrations
- **Google AI (Gemini)**: Primary LLM for narrative generation, session summaries, and world state extraction.
- **AI Image Generation**: Service for generating character portraits.

## 5. Development Workflow
- **Conductor**: Structured development methodology for AI agents and human developers.
- **GitHub Actions**: (Optional) For automated testing and deployment.
